# Easy / Հեշտ (1-3)

1. **Reverse a String / Շրջել տողը**
    Write a function that takes a strng and returns it reversed.
    Գրել ֆունկցիա, որը ստանում է տեղը և վերադարձնում այն շրջված տարբերակով։

2. **Count Vowels / Հաշվել ձայնավորները**
    Count the number of vowels (a, e, i, o, u) in a given string.
    Հաշվել ձայնավորների քանակը տրված տողի մեջ։

3. **Check for Substring / Ենթատողի առկայություն**
    Check if a specific word or character exists within a string.
    Ստուգել՝ արդյոք կոնկրետ բառը կամ նշանը առկա է տողի մեջ։

# Medium / Միջին (4-15)

4. **Palindrome Checker / Պալինդրոմի ստուգում**
    Determine if a string reads the same forwards and backwords.
    Ստուգել՝ արդյոք տողը նույնն է կարդացվում թե սկզբից, թե վերջից։

5. **Capitalize Words / Մեծատառով սկսվող բառեր**
    Capitalize the first letter of every word in a sentence.
    Դարձնել նախադասության յուրաքանչյուր բառի առաջին տառը մեծատառ։

6. **Character Frequency / Տառերի հաճախականություն**
    Count how many times each character appears in a string.
    Հաշվել, թե քանի անգամ է յուրաքանչյուր նշանը հանդիպում տողի մեջ։

7. **Remove Duplicates / Հեռացնել կրկնվողները**
    Remove duplicate characters from a string, keeping only the first occurence.
    Հեռացնել տողից կրկնվող նշանները՝ թողնելով միայն առաջինը։

8. **Word Counter / Բառերի քանակ**
    Count the number of words in a sentence (assume words are separated by spaces)
    Հաշվել բառերի քանակը նախադասության մեջ (համարել, որ բառերը բաժանված են բացատներով)

9. **Anagram Checker / Անագրամի ստուգում**
    Check if two strings are anagrams of each other (contain the same characters in a different order).
    Ստուգել՝ արդյոք երկու տեղերը անագրամ են (պարունակում են նույն նշանները տարբեր հերթականությամբ)

10. **Trancate String / Կրճատել տողը**
    If a string is longer than a specified length, cut it and add "..." at the end.
    Եթե տողն ավելի երկար է, քան նշված չափը, կրճատել այն և վերջում ավելացնել ...։

11. **Find Longest Word / Գտնել ամենաերկար բառը**
    Find and return the longest word in a given sentence.
    Գտնել և վերադարձնել տրված նախադասության ամենաերկար բառը։

12. **Extract Numbers / Հանել թվերը**
    Extract all digits from a string and return them as a new string or number.
    Տողից հանել բոլոր թվանշանները և վերադարձնել որպես նոր տող կամ թիվ։

13. **CamelCase to Snake_case / Ռեգիստրի փոփոխում**
    Convert a string from camelCase format to snake_case.
    Փոխակերպել տողը camelCase ձևաչափից snake_case-ի։

14. **Repeat String / Կրկնել տողը**
    Write a function that repeats a given string n times.
    Գրել ֆունկցիա, որը կրկնում է տրված տողը n անգամ։

15. **Title Case a Sentence / Ձևավորել վերնագիր**
    Make the first letter of each word uppercase and the rest lowercase.
    Յուրաքանչյուր բառի առաջին տառը դարձնել մեծատառ, իսկ մնացածը՝ փոքրատառ։


# Hard / Բարդ (16-20)

16. **First Non-Repeating Character / Առաջին չկրկնվող նշանը**
    Find the first character in a string that does not repeat anywhere else.
    Գտնել տողի առաջին այն նշանը, որը ոչ մի տեղ չի կրկնվում։

17. **String Compression / Տողի սեղմում**
    Compress a string by counting repeated character (e.g. "aaabb" becomes "a3b2").
    Սեղմել տողը՝ հաշվելով կրկնվող նշանները (օր․՝ aaabb-ն դառնում է a3b2)

18. **Longest Palandromic Substring / Ամենաերկար պալինդրոմ ենթատողը**
    Find the longest part of a string that is a palindrome.
    Գտնել տողի ամենաերկար հատվաշը, որը պալինդրոմ է։

19. **Caesar Cliper / Կեսարի ծածկագիր**
    Implement a basic Caesar cipher that shifts letter by a given number.
    Իրականացնել Կեսարի ծածկագիրը, որը տեղաշարժում է տառերը նշված քանակով։

20. **Valid Parentheses / Փակագծերի ճշտություն**
    Check if the parentheses in a string are correctly balanced.
    Ստուգել՝ արդյոք տողում փակագծերը ճիշտ են դասավորված։
